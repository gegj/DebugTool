# 发布说明

## 构建步骤

1. 确认 `csharp/Program.cs` 中版本号已更新：

```csharp
[assembly: AssemblyVersion("1.0.7.0")]
[assembly: AssemblyFileVersion("1.0.7.0")]
[assembly: AssemblyInformationalVersion("1.0.7")]
private const string AppVersion = "1.0.7";
```

2. 执行构建：

```bat
csharp\build.bat
```

3. 构建产物：

```text
csharp\bin\DebugTool.exe
```

## 发布文件

只需要发布：

```text
DebugTool.exe
```

`logo.ico` 已编入 EXE，运行时不需要放在同目录。

当前已编译下载地址：

```text
https://github.com/gegj/DebugTool/raw/refs/heads/main/csharp/bin/DebugTool.exe
```

## 更新 JSON

发布新版本时，同步更新 `latest.json` 并上传到 R2：

```json
{
  "version": "1.0.7",
  "url": "https://github.com/gegj/DebugTool/raw/refs/heads/main/csharp/bin/DebugTool.exe",
  "notes": "更新说明"
}
```

注意：`version` 必须大于旧版程序内置的 `AppVersion`，否则旧版不会提示更新。

## 远程地址

程序内置检测地址：

```text
https://pub-af9d006da1914de6a0c61c166370a2d0.r2.dev/adb/latest.json
```

发布时请同时上传：

```text
DebugTool.exe

```

## 版本检查

构建后可用 PowerShell 检查版本：

```powershell
(Get-Item csharp\bin\DebugTool.exe).VersionInfo
```

重点确认：

```text
FileVersion
ProductVersion
```
