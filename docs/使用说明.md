# 使用说明

## 启动

运行：

```text
DebugTool.exe
```

启动后软件会自动尝试获取 F32/F30PRO 和 REMO 的设备信息。自动获取失败不影响手动操作。

## F32/F30PRO

默认 IP：

```text
192.168.0.1
```

可用操作：

- `获取`：读取设备 IMEI
- `开启 Telnet`：只开启 Telnet
- `开启 Telnet + Debug`：开启 Telnet 和 Debug
- `关闭 Telnet`：关闭 Telnet
- `立即重启设备`：发送重启指令

如果 IMEI 为空，执行开关或重启操作时会自动尝试补全 IMEI。

## REMO

默认 IP：

```text
192.168.100.1
```

可用操作：

- `获取`：读取 REMO IMEI
- `开启 REMO Debug`：自动尝试已知算法开启 Debug
- `立即重启设备`：发送 REMO 重启指令

## 更新检测

软件启动时会后台检测新版本。检测失败不影响主功能。

发现新版本时会提示：

- 当前版本
- 最新版本
- 更新说明
- 是否打开下载地址

## 常见问题

### 无法启动

安装 .NET Framework 4.8 Runtime。

### 获取不到 IMEI

检查：

- 电脑是否连接到设备 WiFi 或同一局域网
- IP 是否正确
- 浏览器是否能访问设备管理地址
- 防火墙或代理是否影响局域网连接

### REMO 获取失败

检查 REMO IP 是否为：

```text
192.168.100.1
```

如设备网段不同，请在界面中改成实际 IP。
